# SKT_ERP_STORES

## Overview

SKT_ERP_STORES is an ERP-based Stores Management System developed to streamline inventory and warehouse operations. The application helps organizations efficiently manage stock movement, purchase transactions, issue transactions, returns, and inventory records through a centralized system.

## Features

* Inventory Management
* Stock Inward & Outward
* Purchase Entry
* Purchase Return
* Issue Entry
* Issue Return
* Stock Transfer
* Barcode Management
* Supplier Management
* Reports & MIS
* User Authentication
* Role-Based Access Control
* Master Data Management

## Prerequisites

* Visual Studio 2019 or later
* Microsoft SQL Server
* .NET Framework (version used by the project)
* IIS or IIS Express

## Installation

1. Clone the repository.

```bash
git clone https://github.com/<your-username>/SKT_ERP_STORES.git
```

2. Open the solution in Visual Studio 2019.

3. Restore the database.

4. Update the database connection string in **Web.config**.

5. Build and run the application.

## Configuration

Update the connection string inside **Web.config** according to your SQL Server instance.
